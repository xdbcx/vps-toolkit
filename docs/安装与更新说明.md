# VPS Toolkit v2.2.2.1 安装与更新说明

## 上传仓库

把压缩包内的内容上传到 GitHub 仓库根目录。

正确结构：

```text
vps-toolkit/
├── install.sh
├── update.sh
├── rollback.sh
├── vps.sh
├── vps-toolkit.sh
├── version
├── config/
├── modules/
└── docs/
```

## 安装

```bash
bash <(curl -sL https://raw.githubusercontent.com/xdbcx/vps-toolkit/main/install.sh)
```

## 呼出

```bash
vt
```

## 常用

```bash
vt check
vt version
vt update
vt rollback
```
