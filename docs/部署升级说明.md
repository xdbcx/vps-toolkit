# VPS Toolkit v2.2.1

## 首次安装

bash <(curl -sL https://raw.githubusercontent.com/xdbcx/vps-toolkit/main/install.sh)

## 常用命令

菜单:
vps-toolkit

巡检:
vps-toolkit check

测速:
vps-toolkit speedtest

更新:
vps-toolkit update

回滚:
vps-toolkit rollback

版本:
vps-toolkit version


## ServerBox快捷命令

健康检查:
vps-toolkit check

更新:
vps-toolkit update


## 发布新版本

1. 修改version
2. Git提交
3. 创建Release
4. VPS执行update


## 更新规则

GitHub保存历史版本。

VPS只运行当前版本。

更新前自动备份到:

/root/vps-toolkit-backup/
